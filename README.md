# cyOS


## 项目简介

这是一个岩岩小天使正在做的操作系统【有参考linux】  
目前正在超级超级初级开发阶段.....

### 已完成

- 进入main函数之前的引导部分
- 屏幕输出相关stdio函数
- 用户态双线程切换
- xmake自动化编译
- vscode调试代码
- 基于lba的扇区读写

### 下一步方向

- 文件系统
- 内存管理
- tui交互界面



## 依赖软件

- xmake 用做构建系统
- mingw 用来编译生成
- bochs 用来调试与执行,推荐 
- qemu 功能与bochs相同，但是速度比bochs快，只不过调试用的是gdb（我不会用）
- 一款好用的开发环境，我用的是vscode，插件一堆一堆的
